# WebGenFree: One Page Business Website Generator

A professional, no-code tool to create stunning one-page business websites in minutes. Build, customize, preview in real-time, and download a production-ready ZIP file of your website—**completely for free**.

## 🚀 Features

*   **Live Real-time Preview**: See your changes instantly as you type.
*   **Section-Wise Editing**: Complete control over Header, Hero, About, Services, and Contact sections.
*   **All Features Free**: Access premium features like image galleries and custom fonts without any cost.
*   **Production-Ready Export**: Generates a clean, standalone ZIP file with all assets embedded.
*   **Fully Responsive**: Your created sites look great on Mobile, Tablet, and Desktop.
*   **No Backend Needed**: Works entirely as a static site, making it easy to host and deploy.
*   **Privacy First**: No login required. Your data stays in your browser.

## 🛠️ How to Use

1.  Open `index.html` to see the landing page.
2.  Click **"Start Building"** to enter the builder.
3.  Fill in your business details into the sidebar panels.
4.  Choose your preferred **Theme** and **Font**.
5.  Click **"Download Your Website"** to get your files instantly.

## 📁 Project Structure

*   `/index.html` - The landing/intro page.
*   `/builder.html` - The main application interface.
*   `/css/landing.css` - Styles for the landing page.
*   `/css/styles.css` - Styles for the builder application.
*   `/js/script.js` - Core logic for the builder and ZIP generation.
*   `/assets/` - Directory for images and icons.

## 🌍 How to Deploy on GitHub Pages

1.  **Create a New Repository**: Create a new repo on GitHub (e.g., `web-builder`).
2.  **Upload Files**: Upload all the files from this directory to the repository.
3.  **Configure GitHub Pages**:
    *   Go to **Settings** -> **Pages**.
    *   In the "Build and deployment" section, under "Source", select **Deploy from a branch**.
    *   Select the **main** branch and **/** (root) folder.
    *   Click **Save**.
4.  **Live Link**: After a minute, your site will be live at `https://your-username.github.io/web-builder/`.
